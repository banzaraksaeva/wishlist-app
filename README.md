# ✨ WishList App

Social wishlist app with real-time updates, group funding and anonymous reservations.

## Features
- 🎁 Create wishlists for any occasion
- 🔗 Share via public link (no registration needed)
- 💰 Anonymous group funding with progress bar
- ⚡ Real-time updates via WebSockets
- 🤖 Auto-fill item details from URL (Wildberries, Ozon)
- 🔒 Owner never sees who reserved or contributed

## Stack
- **Frontend:** Next.js 16 + TypeScript + Tailwind CSS
- **Backend:** FastAPI + Python
- **Database:** PostgreSQL
- **Real-time:** WebSockets
